## ABC 方案切換實例

### 1.1 初始狀態：方案 A（保守穩健）

**起始條件：**
- 初始資金：1000 USDT
- 當前方案：A（48小時翻倍目標）
- 基礎槓桿：20x
- 基礎倉位：30%
- 目標收益率：2.9% per 小時

```
時間：2025-01-01 00:00
資金：1000 USDT
方案：A
狀態：開始運行
```

### 1.2 方案 A → B 升級（連續成功）

**升級條件觸發：**
```
時刻 T+6h：連續 8 筆獲利交易
- 交易 1-8：全部獲利
- 平均收益：3.2% per 交易
- 當前資金：1156 USDT (+15.6%)
- 累計虧損連續：0

✅ 觸發條件：連續獲利 ≥ 8 次
→ 系統自動升級：A → B
```

**升級後變化：**
```python
# 方案 B 參數生效
base_leverage: 20x → 20x（不變）
base_position: 30% → 40%（增加）
target_hourly: 2.9% → 3.9%（提升）
tp_multiplier: 1.0x → 1.2x（放寬）
sl_multiplier: 1.0x → 0.9x（收緊）

新狀態：
時間：T+6h
資金：1156 USDT
方案：B（平衡成長）
```

### 1.3 方案 B → C 升級（高收益率）

**升級條件觸發：**
```
時刻 T+18h：6小時平均收益率達標
- 過去 6h 收益率：28.5% 
- 要求門檻：24%（6h × 4%/h）
- 當前資金：1485 USDT (+48.5%)
- 市場狀態：TRENDING（趨勢明確）

✅ 觸發條件：6小時收益率 ≥ 24%
→ 系統自動升級：B → C
```

**升級後變化：**
```python

新狀態：
時間：T+18h
資金：1485 USDT
方案：C（積極加速）
風險等級：HIGH
```

### 1.4 方案 C → B 降級（連續虧損）

**降級條件觸發：**
```

⚠️ 觸發條件：連續虧損 ≥ 2 次（方案 C）
```

**降級後變化：**

## 動態槓桿調整實例

### 2.1 正常市場狀態（槓桿維持）

**初始狀態：**
```python
# 市場條件
VPIN: 0.25（安全區）
Volatility: 1.8%（低波動）
Signal Strength: 0.72（中等）
Market Regime: CONSOLIDATION

# 槓桿計算
base_leverage = 20x
vpin_multiplier = 1.2x  # VPIN < 0.3
volatility_multiplier = 1.1x  # 波動率 < 2%
signal_multiplier = 1.0x  # 信號強度中等

final_leverage = 20 × 1.2 × 1.1 × 1.0 = 26.4x
→ 限制在 25x（最大值）

實際使用：25x
```

**風險狀態：**
```python

Volatility: 2.5%（正常）
Signal Strength: 0.68
Market Regime: VOLATILE

# 槓桿計算
base_leverage = 20x
vpin_multiplier = 0.5x  # VPIN > 0.7 → 減半

signal_multiplier = 1.0x

final_leverage = 20 × 0.5 × 1.0 × 1.0 = 10x

實際使用：10x（風險保護）
影響：倉位價值減半，風險大幅降低
```

## 動態 TP/SL 調整實例

### 4.1 正常市場（標準 TP/SL）

**場景：**
```python
# 市場條件
ATR: 0.8%（正常波動）
Recent ATR Avg: 0.85%
Signal Duration: 3 分鐘（短）
Current Leverage: 20x

# TP/SL 基準
base_tp = 1.5%（方案 B）
base_sl = 1.0%（方案 B）

# ATR 調整
atr_ratio = 0.8 / 0.85 = 0.94
atr_multiplier = 0.95x  # ATR 比率 < 1.0

# 持續時間調整
duration_multiplier = 1.0x  # 持續時間正常

# 槓桿調整
leverage_multiplier = 1.0x  # 槓桿正常

# 最終 TP/SL
final_tp = 1.5% × 0.95 × 1.0 = 1.425%
final_sl = 1.0% × 0.95 × 1.0 = 0.95%

實際使用：
TP: 1.43%
SL: 0.95%
風險回報比：1.43 / 0.95 = 1.5:1
```

### 4.2 高波動擴大（TP/SL 放寬）

**場景：**
```python
# 市場條件
ATR: 1.8%（高波動）⚠️
Recent ATR Avg: 1.1%
Signal Duration: 8 分鐘（長）
Current Leverage: 15x

# TP/SL 基準
base_tp = 1.5%
base_sl = 1.0%

# ATR 調整
atr_ratio = 1.8 / 1.1 = 1.64
atr_multiplier = 1.3x  # ATR 比率 > 1.5 → 大幅擴大

# 持續時間調整
duration_multiplier = 1.2x  # 持續 > 5 分鐘

# 槓桿調整
leverage_multiplier = 1.0x

# 最終 TP/SL
final_tp = 1.5% × 1.3 × 1.2 = 2.34%
final_sl = 1.0% × 1.3 = 1.3%

實際使用：
TP: 2.34%
SL: 1.3%
風險回報比：2.34 / 1.3 = 1.8:1

說明：高波動時放寬止盈，避免過早離場
```

### 4.3 低波動緊縮（TP/SL 收緊）

**場景：**
```python
# 市場條件
ATR: 0.4%（低波動）
Recent ATR Avg: 0.55%
Signal Duration: 2 分鐘（短）
Current Leverage: 22x

# TP/SL 基準
base_tp = 2.0%（方案 C）
base_sl = 0.8%（方案 C）

# ATR 調整
atr_ratio = 0.4 / 0.55 = 0.73
atr_multiplier = 0.85x  # ATR 比率 < 0.7 → 收緊

# 持續時間調整
duration_multiplier = 1.0x

# 槓桿調整
leverage_multiplier = 0.9x  # 槓桿 > 20x → 收緊 SL

# 最終 TP/SL
final_tp = 2.0% × 0.85 = 1.7%
final_sl = 0.8% × 0.85 × 0.9 = 0.612%

實際使用：
TP: 1.7%
SL: 0.61%
風險回報比：1.7 / 0.61 = 2.8:1

說明：低波動時收緊止損，提高勝率
```

### 4.4 高槓桿保護（SL 大幅收緊）

**場景：**
```python
# 市場條件
ATR: 0.9%
Recent ATR Avg: 0.85%
Signal Duration: 4 分鐘
Current Leverage: 25x（最大）⚠️

# TP/SL 基準
base_tp = 2.0%
base_sl = 0.8%

# ATR 調整
atr_ratio = 0.9 / 0.85 = 1.06
atr_multiplier = 1.0x

# 持續時間調整
duration_multiplier = 1.0x

# 槓桿調整（關鍵）
leverage_multiplier = 0.8x  # 槓桿 = 25x → 強制收緊

# 最終 TP/SL
final_tp = 2.0% × 1.0 = 2.0%
final_sl = 0.8% × 1.0 × 0.8 = 0.64%

實際使用：
TP: 2.0%
SL: 0.64%
風險回報比：2.0 / 0.64 = 3.1:1

爆倉距離：100% / 25x = 4%
止損距離：0.64%
安全倍數：4.0 / 0.64 = 6.25x

說明：高槓桿時必須收緊止損，避免爆倉
```
